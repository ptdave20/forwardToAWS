# forwardToAWS
Forward messages from EC2 back to Amazons abuse department from gmail

# Requirements
credentials.json found here: https://developers.google.com/gmail/api/quickstart/dotnet

# Instructions
- Open the project
- Restore nuget packages
- Copy your credentials.json into your project
- Set the credentials.json to copy always in the properties
- Run
- Accept OAuth
- Wait and smile. You will get email responses the amazon has received the email
